--To try this solution

1) Make sure that mongoDB is installed in your system

2) After installing set environment variable to bin folder

3) md /data

4) md /data/db

5) After that this solution will work.


To try db transactions in command prompt in your local
1) run mongod to start mongo demon server in on cmd window
2) open another command window and run mongo to start mongo shell and then run your transactions
